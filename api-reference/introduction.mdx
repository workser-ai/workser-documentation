## Welcome

There are two ways to build with Workser: use Workser Vibe coding API or use Workser as [Internal tools](https://docs.workser.ai/how-to/internal-tools) for vibe coding. This section will talking about API part only.

Now Workser has 3 main groups of API: Project, Task, Publish.

<Card title="Project API" icon="file-binary">
  Get data, Manage, and create project
</Card>

<Card title="Task API" icon="list-check">
  Get data, manage, and create task
</Card>

<Card title="Publish API" icon="cloud-arrow-up">
  Manage custom domain on workser.app and deployment alias
</Card>
